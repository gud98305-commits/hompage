# aeae Pop-up Store — 프로젝트 문서

## 프로젝트 개요

**브랜드**: aeae (에이에이) — 한국 패션 브랜드 (ae-ae.kr)
**프로젝트 유형**: 온라인 팝업스토어 인터랙티브 웹 경험
**배포 환경**: Netlify (Serverless Functions 활용)
**타겟 시장**: 한국 + 일본 (이중언어: 일본어 메인, 한국어 서브)
**브랜치**: `popup_store`

---

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | Vanilla HTML/CSS/JS (프레임워크 없음) |
| 백엔드 | Netlify Serverless Functions |
| AI API | OpenAI GPT-4o-mini (Netlify Function 프록시) |
| 외부 라이브러리 | QRCode.js, html2canvas |
| 폰트 | Caveat, Noto Serif JP, VT323, Cormorant Garamond, Courier Prime |
| 호스팅 | Netlify |

---

## 페이지별 상세 분석

### 1. index.html — 메인 팝업스토어 경험

**컨셉**: RPG 스타일의 "방 정리" 인터랙티브 경험

**사용자 흐름**:
1. **인트로 화면**: 로고 + "aeaeの部屋へ、ようこそ。" + ENTER 버튼 (문이 열리는 애니메이션)
2. **방 탐색**: 24개 아이템이 흩어진 방 — 클릭/드래그로 아이템 제거
3. **가방 발견**: 모든 아이템 제거 후 가방(thebag) 발견 → 클릭
4. **가방 클로즈업** (STEP 1): 가방 확대 이미지
5. **가방 오픈** (STEP 2): 가방을 열면 빨간 봉투 등장
6. **봉투 열기** (STEP 3): 3가지 이벤트 메뉴 카드 표시
   - **Lucky Roulette** → lookbook.html
   - **Dear aeae** → receipt.html
   - **Shop Now** → ae-ae.kr (외부 쇼핑몰)

**핵심 기능**:
- RPG 대화 시스템 (타이핑 애니메이션 + 순차 대사)
- 드래그 & 클릭으로 아이템 제거 (터치 지원)
- 아이템 카운터 ("✦ N個残ってる")
- `?menu=1` 쿼리 파라미터로 메뉴 화면 직행 가능

### 2. lookbook.html — Lucky Roulette (스타일 룰렛)

**컨셉**: 슬롯머신 형태의 코디 추천 시스템

**사용자 흐름**:
1. SPIN 버튼 클릭 → 슬롯머신 회전 (TOP / BOTTOM / ACC 3릴)
2. 30개 룩 중 랜덤 선택 → 전체 코디 이미지 + 개별 아이템 카드 표시
3. AI가 스타일 분석 + 오늘의 패션 운세 생성 (GPT-4o-mini)
4. 각 아이템별 실제 쇼핑몰 구매 링크 제공

**데이터**:
- 30개 룩 (Look #01 ~ #30)
- 각 룩: 전체 이미지(a_N.png), 상의(a_N_top.png), 하의(a_N_bottom.png), 액세서리(a_N_item.png)
- 10개 룩은 액세서리 없음 (NO_ACC: 3, 5, 6, 7, 8, 10, 18, 19, 21, 28)
- 각 아이템에 ae-ae.kr 상품 페이지 URL 매핑 (PURLS 객체)

**AI 기능**:
- Netlify Function (`/.netlify/functions/openai`)을 통한 OpenAI API 호출
- 일본어 캐주얼체로 스타일 분석 + 패션 운세 생성
- API 실패 시 폴백 메시지 표시

### 3. receipt.html — Dear aeae (AI 상담소)

**컨셉**: 레트로 컴퓨터 + 프린터에서 영수증 형태로 위로 메시지 출력

**사용자 흐름**:
1. 이름 입력 + 오늘의 고민 작성
2. PRINT RECEIPT 클릭 → AI가 위로 메시지 생성
3. 레트로 컴퓨터 프린터에서 영수증이 인쇄되듯 슬라이드 다운
4. 영수증에 포함: 감정 분석, 위로 메시지 (일/한), 할인코드, QR코드, 바코드
5. SAVE (이미지 저장) / INSTAGRAM (공유) / NEW (다시 작성)

**AI 프롬프트 설계**:
- 브랜드 가치 반영: Neverland(순수함), Keepsake(소중한 감정), Touch(이해받는 안도감)
- 친한 친구처럼 따뜻하고 진심 어린 톤 (상담사/치료사 톤 금지)
- 일본어 + 한국어 이중 메시지 출력

**영수증 요소**:
- aeae 로고 + "keepsake" 브랜드
- 이름, 날짜, 감정 라인
- 고민 원문 (인용 형태)
- AI 위로 메시지 (일본어 + 한국어)
- 할인코드 (KEEPAEAE_15, 15% OFF)
- QR코드 (ae-ae.kr 링크)
- 바코드 (랜덤 생성)
- html2canvas로 이미지 저장 기능

## 마케팅 전략 분석

### 1. 고객 여정 설계 (Customer Journey)

```
[인지] 인트로 → [흥미] 방 정리 게임 → [몰입] 가방/봉투 스토리 → [전환] 3가지 CTA
                                                                    ├─ Lucky Roulette (상품 노출)
                                                                    ├─ Dear aeae (감정적 연결)
                                                                    └─ Shop Now (직접 구매)
```

- **Awareness → Interest → Engagement → Conversion** 퍼널이 게임/스토리텔링 안에 자연스럽게 녹아있음
- 사용자가 "방을 정리하는 행위" 자체가 브랜드 세계관에 몰입하게 만드는 장치
- 인트로 → 탐색 → 발견의 과정이 "보물찾기" 감성을 자극하여 이탈률 최소화

### 2. 감성 마케팅 (Emotional Marketing)

**브랜드 감성 키워드**: 따뜻함, 노스탤지아, 손편지, 꿈결, 친밀감

| 요소 | 마케팅 효과 |
|------|------------|
| RPG 대화 시스템 | 브랜드를 "친구"로 포지셔닝 — 일방적 광고가 아닌 대화형 관계 |
| 일본어+한국어 이중언어 | 일본 시장 공략 + 글로벌 감성 연출 |
| 빨간 봉투(편지) | "브랜드가 나에게 보내는 특별한 편지" — 1:1 감정 연결 |
| 레트로 컴퓨터 UI | Z세대 뉴트로 트렌드 + 아날로그 감성으로 차별화 |
| 영수증 형태 위로 메시지 | SNS 공유 유도 (영수증 사진 = 바이럴 콘텐츠) |

### 3. 게이미피케이션 (Gamification)

| 요소 | 게이미피케이션 원리 | 마케팅 목적 |
|------|-------------------|------------|
| 24개 아이템 클릭 제거 | 수집/완료 욕구 자극 (Progress) | 페이지 체류시간 극대화 |
| 아이템 카운터 | 목표 가시화 (Feedback Loop) | 이탈 방지 |
| 가방 발견 메시지 | 보상감 (Reward) | 긍정적 감정 → 브랜드 호감도 상승 |
| 봉투 열기 시퀀스 | 서프라이즈/언박싱 (Surprise) | 호기심 → 전환 유도 |
| 슬롯머신 룰렛 | 랜덤성 + 반복 욕구 (Variable Reward) | 여러 상품 자연 노출 |
| 스티커 비 이펙트 | 축제/보상 연출 (Celebration) | 공유 욕구 자극 |

### 4. 전환(Conversion) 전략

#### Lucky Roulette → 구매 전환
- 30개 룩 → 랜덤 노출 = 카탈로그 전체를 자연스럽게 탐색하게 만듦
- 각 아이템별 **직접 구매 링크** (ae-ae.kr 상품 페이지)로 즉시 전환 가능
- "운명의 코디" 프레이밍 → 구매 의사결정에 "재미" 요소 추가
- AI가 생성한 스타일 분석 → 상품의 부가 가치 + 설득력 강화
- "もう一度 ↺" (다시 돌리기) → 반복 플레이로 다양한 상품 노출

#### Dear aeae → 브랜드 로열티 + 재방문
- AI 위로 메시지 → **감정적 가치** 제공 (상품 판매 이전의 신뢰 구축)
- 할인코드 자동 포함 (KEEPAEAE_15, 15% OFF) → 즉각적 구매 인센티브
- QR코드 → ae-ae.kr 바로 이동
- 이미지 저장 + Instagram 공유 → **UGC(User Generated Content) 유도**
- 영수증 디자인 자체가 공유할만한 비주얼 → 바이럴 마케팅

### 5. SNS 바이럴 전략

| 콘텐츠 | 공유 유도 포인트 |
|--------|----------------|
| 영수증 이미지 | "나만의 위로 영수증" → Instagram 스토리 공유 소재 |
| 룰렛 결과 | "오늘의 운명 코디" → 친구 태그 + 공유 |
| 방 정리 게임 | "가방 찾기 게임" → 재미 요소로 링크 공유 |
| AI 운세 | "패션 운세" → 호기심 유발 공유 |

**바이럴 루프**: 사용자 → 콘텐츠 생성(영수증/룰렛) → SNS 공유 → 새 사용자 유입 → 반복

### 6. 일본 시장 공략 전략

- **메인 언어 일본어** (`lang="ja"`) — 일본 고객 우선 타겟팅
- 일본어 대화 + 한국어 자막 형식 → "한국 브랜드의 일본 진출" 포지셔닝
- 일본 서브컬처 감성 (RPG 대화창, 레트로 컴퓨터) 활용
- Noto Serif JP, Noto Sans JP 등 일본어 최적화 폰트 사용
- AI 프롬프트도 일본어 출력 우선 ("ため口" = 반말 친근체 지정)

### 7. 브랜드 세계관 구축

**aeae 세계관 3대 키워드**:
1. **Neverland** — 어른이 되어도 단단해지지 않는, 부드러운 마음
2. **Keepsake** — 복잡해도 간직할 가치가 있는 감정들
3. **Touch** — 진짜로 이해받는 순간의 안도감

이 세계관이 각 페이지에 자연스럽게 녹아있음:
- index.html: "방"이라는 개인 공간 → Neverland (어린 시절의 방)
- lookbook.html: "운명의 코디" → Keepsake (오늘 하루의 작은 운명)
- receipt.html: "위로 편지" → Touch (이해받는 경험)

| 디자인 요소 | 효과 |

### 9. 데이터 수집 및 CRM 관점

- **Dear aeae** 입력 데이터: 이름 + 고민 내용 → 고객 감정/니즈 인사이트 (현재 서버 저장 미구현)
- **룰렛 결과**: 어떤 룩이 많이 조회되는지 → 인기 상품 트래킹 가능 (현재 미구현)
- **할인코드 사용률**: KEEPAEAE_15 사용 추적 → 팝업 → 구매 전환율 측정
